# 🇧🇷 Sobre o Projeto

Projeto desenvolvido na disciplina de **Fundamentos da Programação** durante o primeiro semestre de Engenharia de Software.

A aplicação consiste em um sistema de gerenciamento de contatos desenvolvido em **linguagem C**, utilizando conceitos de programação estruturada, modularização e manipulação de dados via terminal.

# ✨ Funcionalidades

- Cadastro de contatos
- Busca de registros
- Edição de informações
- Remoção de contatos
- Validação básica de entradas

# ⚙️ Tecnologias

- Linguagem C
- Programação Estruturada
- Interface CLI

# 🇺🇸 About the Project

This project was developed during the **Programming Fundamentals** course in the first semester of Software Engineering.

The application is a contact management system built in **C language**, applying structured programming concepts, modularization, and terminal-based data handling.

# ✨ Features

* Add contacts
* Search records
* Edit information
* Remove contacts
* Basic input validation

# ⚙️ Technologies

* CLI Interface
